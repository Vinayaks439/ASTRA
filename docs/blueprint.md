# **App Name**: SellerFlow AI

## Core Features:

- SKU Risk Dashboard: View real-time risk scores (Composite, Price Gap, Stock Coverage, Demand Trend, Margin Proximity) for all SKUs in a sortable and filterable dashboard table.
- Live AI Summary Insights: A dynamic panel providing natural language summaries of key operational insights, SKU risk posture, and urgent actions, generated by an AI tool based on deterministic metrics.
- AI-Assisted Price Adjustment: Receive intelligent price adjustment recommendations (increase, decrease, hold) for SKUs based on current risk profile and configured guardrails, generated by an AI tool.
- AI-Driven PO Recommendations: Automatically generate purchase order recommendations for low-inventory SKUs based on vendor MOQ and sales velocity, with configurable thresholds via an AI tool.
- Guardrail Enforcement & Agent Autonomy: The system enforces pre-defined thresholds and guardrails, allowing autonomous agent actions only within set limits, blocking actions if guardrails are violated.
- Exception Action Ticketing: Automatically creates exception tickets for agent-recommended actions that exceed autonomous thresholds, requiring human review and explicit approval from the seller.
- Configurable Seller Settings: Adjust key operational thresholds, activate/deactivate features like PO recommendations, configure WhatsApp notifications, and personalize the dashboard theme (light/dark mode).

## Style Guidelines:

- Primary interactive color: A purposeful, deep blue (#6189DB) to signify focus and control, contrasting clearly against a dark theme background.
- Background color: A muted, very dark blue-gray (#20242A), creating a sophisticated 'command center' ambiance suitable for long operational sessions in a dark mode environment.
- Accent color: A vibrant, clear blue-cyan (#77D4ED) for calls-to-action, highlights, and urgent indicators, providing high visibility and modernity.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired command center feel, used for primary titles and critical alerts. Body text font: 'Inter' (sans-serif) for high legibility and efficient consumption of information-dense data displays and tables.
- Utilize a sleek, minimalist icon set to visually communicate actions, distinct risk statuses (Critical, Warning, Healthy), and various operational alerts, enhancing scanability and decision speed without visual clutter.
- Employ a desktop-first, multi-column grid layout featuring a prominent, sticky AI Summary panel at the top for immediate insights, a main action-oriented SKU table for detailed review, and a persistent left-hand navigation. Ensure fluid transitions for drawers and overlays for secondary data.
- Implement subtle, performance-optimized micro-animations for interactive elements, view transitions, and feedback on user actions. This ensures a fluid, responsive, and polished user experience congruent with a modern 'command center' interface.
